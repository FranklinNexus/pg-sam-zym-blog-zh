# 别跟 Corp Dev 谈

> 原文：<https://paulgraham.com/corpdev.html>
> 来源：[paulgraham.com](https://paulgraham.com/)
> 英文标题：Don't Talk to Corp Dev
> 年代：2015 年 1 月

**Corp Dev**（Corporate Development，企业发展/并购部门）是公司内部负责**收购其他公司**的团队。若有人在跟你谈，原因就是这个——不管你有没有意识到。

### 什么时候可以谈

除非（a）你**现在就想卖公司**，且（b）拿到可接受报价的概率足够高，否则跟 Corp Dev 谈通常是**错误**。

实践中，初创公司只应在**极好或极差**时跟他们谈。极差意味着公司快死了，谈谈也无妨。极好时你可以安全地谈——双方都知道价格必须很高，他们若稍有浪费你时间的迹象，你也有底气让他们滚。

### 危险区：中间地带

危险在于**中间状态**的公司——增长很快、但时间还不够长、体量还不够大的年轻公司。成立不到一年、前景不错的公司，**通常也不该**跟 Corp Dev 谈。

但创始人**一再犯这个错**。Corp Dev 想见面时，他们会想：至少该弄清对方要什么；又不想得罪大公司而拒绝会面。

我告诉你他们想要什么：**谈收购你。**「Corp Dev」这个头衔就是这个意思。同意见面之前问自己：**「我们现在想卖公司吗？」**若答案是否，就说：「抱歉，我们正专注把公司做大。」对方不会介意；大公司的创始人也不会——若有什么，他们会更敬重你。你会让他们想起自己：他们当年也没卖，所以现在才有能力买别人。[1]

### 明知不该，为何还去

多数被 Corp Dev 联系上的创始人**已经知道**这意味着什么。明知不想卖，仍去开会。为何？与大多数创始人错误一样：**否认 + 一厢情愿**。

有人想买你，很**受宠若惊**。万一报价高得离谱呢？至少看看？**不。**

若他们会立刻用邮件发 offer，你不妨打开。但 Corp Dev 的对话**不是这么运作的**。若有 offer，往往在漫长、令人难以置信的干扰过程**末尾**才出现；若报价「惊喜」，通常是**惊喜地低**。

### 最糟的干扰

初创公司最承受不起的就是**干扰**。Corp Dev 的对话是最糟的一类：不只消耗[注意力](https://paulgraham.com/top.html)，还**打击士气**。艰苦过程中活下去的诀窍之一，是别停下来想自己有多累——你进入某种**心流**。[2]

想象马拉松跑到第 20 英里，有人跑过来说：「你一定很累，要不要停下来歇歇？」跟 Corp Dev 谈就像这样，**更糟**——因为「停下来」的建议，会在你脑中与想象中他们会出的**天价**绑在一起。那你就真有麻烦了。

### 他们的战术

Corp Dev 喜欢**扭转局面**：让你拼命说服他们买，而不是他们说服你卖。令人惊讶的是，他们**常常成功**。

这是极滑的斜坡，涂满最能作用于创始人心理的力量，还有一位全职把你推下去的专业人士。推你下去的战术通常相当**粗暴**。他们的工作就是买公司，还不能挑标的；绩效只看**多便宜**买下你，有野心的人会不择手段。例如几乎总以**低价开场**试探——即便你不接受，低价也会让你士气低落、更易操纵。这还算**最无辜**的。

谈妥价格、以为成交了，他们回来却说老板否决，最多只肯付商定价格的一半——**经常发生**。若你觉得投资人有时行为恶劣，跟 Corp Dev 比**不算什么**，即便在整体友善的公司里亦然。

我曾向 Google 的朋友抱怨他们对 YC 初创耍的阴招。「不作恶怎么了？」「我觉得 Corp Dev 没收到备忘录。」他答。

并购对话里的战术，可能与你经历过的、相对[正派](https://paulgraham.com/mean.html)的硅谷世界截然不同——仿佛老式强盗大亨商业世界的一块基因嵌进了创业世界。[3]

### 洛克菲勒的「第一口酒」

最简单的自保，是用约翰·D·洛克菲勒的伎俩——他祖父是酒鬼，他用这招避免自己也成为酒鬼。他曾对主日学班的男孩说：

> 你们知道我为什么从没变成酒鬼吗？因为我从没喝过第一口酒。

**你现在想卖公司吗？**不是最终，是**现在**。若不想，就别参加第一次会面。他们不会介意；你也能免于初创公司可能遭遇的最糟经历之一。

若你确实想卖，另有一套[技巧](https://justinkan.com/the-founders-guide-to-selling-your-company-a1b2025c9481)。但创始人在 Corp Dev 上最大的错误，不是准备好时谈得差，而是**还没准备好就谈**。若你只记住本文标题，第一年关于并购该知道的，你已经知道大半。

## 注释

[1] 不是说永远不该卖，而是心里要清楚想不想卖，别被操纵或一厢情愿带着，比本打算更早地卖。

[2] 初创公司像多数竞技运动，手头任务几乎替你挡住疲惫；一失去这层保护（例如终场哨响），疲劳像浪打来。跟 Corp Dev 谈等于**赛中**就让自己感受它。

[3] 公平说，Corp Dev 表面的不当行为被放大，因为他们是大组织的门面，而组织常连自己想要什么都说不清。收购方在收购上可能出人意料地优柔寡断，传到你这层时，反复无常与不诚实已难以区分。
