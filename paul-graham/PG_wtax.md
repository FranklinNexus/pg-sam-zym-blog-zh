# 财富税建模

> 原文：<https://paulgraham.com/wtax.html>
> 来源：[paulgraham.com](https://paulgraham.com/)
> 英文标题：Modeling a Wealth Tax
> 年代：2020 年 8 月

一些政客提议在所得税和资本利得税之外再征**财富税**。下面用简单模型估算不同税率对创业创始人意味着什么。

假设你二十多岁创办成功初创，再活 60 年。财富税会吃掉你多少股权？

### 无起征点、资产全征

若对所有资产征税，计算很直接：年税率 1% 意味着每年保留 99% 股份。60 年后剩余比例为 0.99^60 ≈ **0.547**——即一生约 **45%** 股权被政府拿走。（股份变少不等于净资产变差，除非每股增值低于财富税率。）

| 年财富税率 | 60 年后政府累计拿走 |
|-----------|---------------------|
| 0.1% | 6% |
| 0.5% | 26% |
| 1.0% | 45% |
| 2.0% | 70% |
| 3.0% | 84% |
| 4.0% | 91% |
| 5.0% | 95% |

### 有起征点时

财富税通常有起征门槛。差别多大？需假设初始股权价值与增长路径。

设初始股权价值 **200 万美元**，轨迹为：价值先 **3 倍/年** 连 2 年，再 **2 倍/年** 连 2 年，再 **50%/年** 连 2 年，之后按典型上市公司增速（此处取 **8%/年**）增长。起征门槛 **5000 万美元** 时：

| 年财富税率 | 60 年后政府累计拿走 |
|-----------|---------------------|
| 0.1% | 5% |
| 0.5% | 23% |
| 1.0% | 41% |
| 2.0% | 65% |
| 3.0% | 79% |
| 4.0% | 88% |
| 5.0% | 93% |

税率看似不高，效果却极猛：**2% 财富税 + 5000 万门槛**，仍可能拿走成功创始人约 **三分之二** 的股权。原因是财富税对**同一笔钱年复一年**征收：所得税每年只对当年收入征一次；你持有资产 60 年，财富税就对同一资产征 60 次——**复利式叠加**。

---

**注释**

**[1]** 实践中，部分 8% 会以股息形式实现，发息时按所得税处理；故本模型对创始人已是**最乐观**情形。
