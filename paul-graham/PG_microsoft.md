# 微软已死

> 原文：<https://paulgraham.com/microsoft.html>
> 来源：[paulgraham.com](https://paulgraham.com/)
> 英文标题：Microsoft is Dead
> 年代：2007 年 4 月（几天前我突然意识到**微软**已经死了。我正在与一位年轻的初创公司创始人谈论**谷歌**与**雅虎**的不同之处。我说过，**雅虎**从一开始就因为对）

### 阴影消失

**微软**的恐惧而扭曲了。这就是为什么他们将自己定位为“媒体公司”而不是科技公司。然后我看着他的脸，发现他不明白。就好像我告诉他八十年代中期女孩们有多么喜欢巴里·曼尼洛。巴里是谁？**微软**？他什么也没说，但我看得出他不太相信有人会害怕他们。从 80 年代末开始，**微软**给软件世界蒙上了阴影近 20 年。我记得他们之前是IBM。我基本上忽略了这个影子。我从未使用过微软软件，所以它只是间接地影响了我——例如，在我从僵尸网络收到的垃圾邮件中。而且因为我没有注意，所以我没有注意到影子什么时候消失的。但现在已经消失了。我能感觉到。甚至没有人再害怕微软了。他们仍然赚了很多钱——就这一点而言，IBM 也是如此。但它们并不危险。微软什么时候死的，死的是什么？我知道它们在 2001 年就显得很危险，因为我当时写了一篇文章，讲述它们并不像看上去那么危险。我猜他们到 2005 年就已经死了。我知道当我们创办 **Y Combinator** 时，我们并不担心微软会成为我们资助的初创公司的竞争对手。事实上，我们甚至从未邀请他们参加我们为初创公司组织的向投资者展示的演示日。我们邀请了**雅虎**和**谷歌**以及其他一些互联网公司，但我们从来没有邀请过微软。那里甚至没有人给我们发过电子邮件。他们在不同的世界。是什么杀死了他们？我认为有四件事在 2000 年代中期同时发生。最明显的是**谷歌**。镇上只能有一个大人物，而他们显然就是这个大人物。无论从好还是坏的意义上来说，**谷歌**都是目前为止最危险的公司。微软最多只能在之后步履蹒跚。谷歌什么时候领先了？人们倾向于将其推迟到 2004 年 8 月进行首次公开​​募股，但当时他们并没有设定辩论条款。我想说他们在 2005 年取得了领先地位。Gmail 是让他们陷入困境的因素之一。 Gmail 表明他们可以做的不仅仅是

### 四件事

搜索。 Gmail 还展示了如果您利用后来被称为“Ajax”的技术，您可以使用基于网络的软件做多少事情。这就是微软死亡的第二个原因：每个人都可以看到桌面已经结束了。现在看来，应用程序将不可避免地存在于网络上——不仅仅是电子邮件，还有一切，甚至包括 Photoshop。甚至微软现在也看到了这一点。讽刺的是，微软无意中帮助创建了 Ajax。 Ajax 中的 x 来自 XMLHttpRequest 对象，它允许浏览器在显示页面时在后台与服务器通信。 （最初与服务器通信的唯一方法是请求一个新页面。）XMLHttpRequest 是 Microsoft 在 90 年代末创建的，因为他们需要它用于 Outlook。他们没有意识到它对很多其他人也很有用——事实上，对任何想要让网络应用程序像桌面应用程序一样工作的人来说都是有用的。 Ajax 的另一个关键组件是 Javascript，一种在浏览器中运行的编程语言。微软看到了 Javascript 的危险，并试图尽可能长时间地阻止它的发展。 [1] 但最终开源世界赢了，他们生产了 Javascript 库，这些库在资源管理器的破碎上生长，就像树在铁丝网上生长一样。微软死亡的第三个原因是宽带互联网。任何关心的人现在都可以快速访问互联网。连接服务器的管道越大，您对桌面的需求就越少。棺材上的最后一颗钉子来自苹果公司。感谢 OS X，苹果以一种在科技领域极为罕见的方式起死回生。 [ 2 ] 他们的胜利是如此彻底，以至于当我遇到一台运行 Windows 的计算机时，我感到很惊讶。我们在 **Y Combinator** 资助的几乎所有人员都使用苹果笔记本电脑。创业学校的观众也是如此。现在所有的电脑用户都使用 Mac 或 Linux。 Windows 是给老奶奶们用的，就像 90 年代 Mac 机一样。因此，不仅桌面不再重要，而

### 能否复活

且关心计算机的人也不会使用微软的桌面。当然，苹果在音乐领域也与微软展开了竞争，电视和手机也即将问世。我很高兴微软死了。他们就像尼禄或康茂德——只有继承的权力才能使你变得邪恶。因为请记住，微软的垄断并不是从微软开始的。他们从 IBM 那里得到了它。从 20 世纪 50 年代中期到 2005 年左右，软件行业一直处于垄断状态。实际上，从它存在的整个时期来看，都是如此。 “Web 2.0”如此令人兴奋的原因之一是，无论有意无意，人们都认为这个垄断时代可能终于结束了。当然，作为一名黑客，我情不自禁地思考如何修复损坏的东西。微软有什么办法可以卷土重来吗？原则上是的。要了解具体情况，请想象两件事：(a) 微软现在手头有多少现金，以及 (b) 十年前，拉里和谢尔盖在所有搜索引擎中四处奔走，试图以 100 万美元的价格出售谷歌的想法，但遭到所有人的拒绝。令人惊讶的事实是，按照像微软这样富有的公司的标准，聪明的黑客——危险的聪明黑客——可以非常便宜地获得。他们不能再雇佣聪明人了，但他们可以购买任意数量的人才，而价格只需高出一个数量级。因此，如果他们想再次成为竞争者，他们可以这样做：购买所有优秀的“Web 2.0”初创公司。他们可以以比购买 Facebook 更低的价格获得几乎所有这些服务。将它们全部放在**硅谷**的一栋建筑物中，周围有铅屏蔽，以保护它们不与雷德蒙德发生任何接触。我觉得提出这个建议是安全的，因为他们永远不会这样做。微软最大的弱点是他们仍然没有意识到自己有多糟糕。他们仍然认为他们可以在内部编写软件。按照桌面世界的标准，也许他们可以。但那个世界几年前就结束了。我已经知道这篇文章会引起什么反应。一半的读者会说，微软仍然是一家利润丰厚的公司，我应该更加谨慎地根据我们孤立的小“Web 2.0”泡沫中少数人的想法得出结论。另一半，年轻的一半，会抱怨这是旧闻。
